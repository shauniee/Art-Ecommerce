# Basic Art E-Commerce Website

## Description

This website allows users to login to their account and add art pieces to their carts.

## Features

- Login Validation
- User authentication
- Product browsing
- Add to cart

## Technologies Used

- HTML
- Bootstrap
- CSS
- JavaScript
- JQuery
